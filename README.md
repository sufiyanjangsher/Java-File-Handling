# Java File Handling

This repository contains Java programs and examples related to File Handling. The purpose of this repository is to understand how Java interacts with files for reading, writing, creating, and managing data stored on a computer.

## Topics Covered

### 1. Introduction to File Handling

* What is File Handling?
* File Class
* Creating Files
* Checking File Existence

### 2. Reading Files

* FileReader
* BufferedReader
* Reading Character Data
* Reading Line by Line

### 3. Writing Files

* FileWriter
* BufferedWriter
* Writing Data to Files
* Appending Data to Existing Files

### 4. File Operations

* Create File
* Read File
* Write File
* Delete File

### 5. File and Exception Handling

* Handling FileNotFoundException
* Handling IOException
* Using try-catch with File Operations

### 6. Practice Programs

* Create and Write to a File
* Read Data from a File
* Append Data to a File
* Copy File Content
* Simple Notes Storage Program

## Concepts Practiced

* File Management
* Data Persistence
* Input/Output Streams
* Exception Handling
* Reading and Writing Data

## Language

Java

## Learning Goal

To understand how data can be stored and retrieved using files in Java and to build a foundation for creating applications that work with persistent data.
